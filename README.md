# Crop Damage Detection AI

This is an AI-based project for detecting crop diseases using machine learning and image processing.

## Features
- Crop disease detection
- AI-based prediction
- Web interface

## SMS setup
Password reset SMS uses MSG91. Add these environment variables before trying the forgot-password flow:

- `MSG91_AUTH_KEY`
- `MSG91_SENDER_ID`
- `MSG91_ROUTE` (optional, defaults to `4`)
- `MSG91_COUNTRY` (optional, defaults to `91`)
